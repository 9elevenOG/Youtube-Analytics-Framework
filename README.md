 # YouTube Analytics Framework

A scalable, open-source content analytics framework with MCP (Model Context Protocol) integration.

## Features

- YouTube Analytics Data Collection
- AI-Powered Content Insights
- Sentiment Analysis
- Thumbnail Analysis
- Competitor Benchmarking
- MCP Server Integration
- Web Dashboard

## Setup

1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate virtual environment: `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and configure your API keys
6. Run the application: `python -m src.main`
